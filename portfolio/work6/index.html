instalacion Hugo + tema

  • Para activar el link de la pagina web, lo realizamos en Github.

Pasos

  1. Descargar el Tema de hugo theme

  2. Crear en el terminal una carpeta llamada public, abrir ambas carpetas , copiar la carpeta del tema descargado y pegarla en la carpeta Hugo/ theme/. para luego remplazar el contenido de la carpeta examplesite a la carpeta hugo.

  3. En el sistema ejecutamos Hugo server. Para optener el link de nuestro portafolio en la web

  4. Abrir visual, editar, cargar y editar contenido de la web para luego ejecutar Comando Hugo el cual nos crea una carpeta dentro de hugo llamada public la cual en cinjunto con github nos permite hacer un commit el cual nos otorga el push origin para poder subir el portafolio a la web.

  5. abrir la direccion en el navegador.
